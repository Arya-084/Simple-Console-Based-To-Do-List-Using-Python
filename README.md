# Simple-Console-Based-To-Do-List-Using-Python
Python To-Do List App using OOP with features to add, view, and complete tasks through a simple console interface.
To-Do List Application


A beginner-friendly Python project that helps users manage daily tasks efficiently. The application is built using Object-Oriented Programming (OOP) concepts and provides a simple menu-driven interface.


Features:
  1.Add new tasks
  2.View all tasks
  3.Mark tasks as completed
  4.User-friendly console menu
  5.Object-Oriented Programming implementation


Technologies Used:
  1.Python 3
  w.OOP (Classes and Objects)


Concepts Covered:
  1.Classes and Objects
  2.Constructors (init)
  3.Methods
  4.Lists
  5.Functions
  6.Loops
  7.Conditional Statements


How to Run:

1.Clone the repository.
2.Open the project in VS Code or any Python IDE.
3.Run the Python file: python file_name.py
4.Follow the on-screen menu instructions.

Sample Output:
===== TO-DO LIST MENU =====
1. Add Task
2. View Tasks
3. Mark Task as Completed
4. Exit
