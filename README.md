# Simple-Console-Based-To-Do-List-Using-Python
Python To-Do List App using OOP with features to add, view, and complete tasks through a simple console interface.
To-Do List Application

A beginner-friendly Python project that helps users manage daily tasks efficiently. The application is built using Object-Oriented Programming (OOP) concepts and provides a simple menu-driven interface.
Features:
  Add new tasks
  View all tasks
  Mark tasks as completed
  User-friendly console menu
  Object-Oriented Programming implementation
Technologies Used:
  Python 3
  OOP (Classes and Objects)
Concepts Covered:
  Classes and Objects
  Constructors (init)
  Methods
  Lists
  Functions
  Loops
  Conditional Statements
How to Run:

Clone the repository.
Open the project in VS Code or any Python IDE.
Run the Python file: python file_name.py
Follow the on-screen menu instructions.
Sample Output:
===== TO-DO LIST MENU =====
1. Add Task
2. View Tasks
3. Mark Task as Completed
4. Exit
